# Fluxo
Este é o fluxo principal, descrito de forma geral, que vou me orientar na construção do MVP.

## Etapas
- Usuário loga e faz apostas nos jogos disponíveis.
- Sistema trava as apostas minutos antes do jogo começar.
- Admin insere o resultado final após o apito do juiz.
- Sistema calcula os pontos de todos e atualiza o ranking.
